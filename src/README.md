# DARTS/ISAS SPICE Datasource

Data source plugin that gets data from spice

## Data Source Setting

### SPICE Kernels: URL of the dependent kernel

Specifies the Kernel to use for the calculation. Multiple Kernels can be set.

e.g.

* http://example.com/kernels/lsk/naif0012.tls
* http://example.com/kernels/spk/de432s.bsp

Note: The server providing the kernel must respond to the Preflight Request.
